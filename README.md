# 📌 Projeto API de Tarefas

Uma API REST desenvolvida com **Kotlin** e **Spring Boot** para gerenciamento de tarefas.

## ✅ Funcionalidades

- Criar tarefas
- Listar todos os tarefas
- Buscar tarefas por ID
- Atualizar tarefas
- Deletar tarefas

## 🛠 Tecnologias utilizadas

- Kotlin
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
